# 📖 Deskripsi

Sistem Informasi Manajemen Cuti Karyawan merupakan aplikasi berbasis web yang dikembangkan untuk membantu proses pengajuan, verifikasi, dan persetujuan cuti secara digital sehingga proses administrasi menjadi lebih cepat, efisien, dan terdokumentasi dengan baik.

---

# 🎯 Tujuan Project

- Mempermudah pengajuan cuti secara online.
- Mempermudah HRD mengelola data cuti.
- Mempermudah Direktur menyetujui atau menolak pengajuan.
- Mengurangi penggunaan dokumen manual.
- Menampilkan status cuti secara real-time.

---

# ✨ Fitur

- 🔐 Login Multi Role
